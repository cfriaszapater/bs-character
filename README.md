BS character frontend, built with react + redux.

## Development

To run locally:

### Build

```sh
npm install
```

### Test

```sh
npm test
```

### Run

```sh
npm start
```

### Deploy to production

To deploy to production (having an heroku account and git heroku remote set to point to it):

```sh
git push heroku master && heroku open
```

View logs:

```sh
heroku logs --tail
```

Config:

```sh
heroku config
```
